# DecodeLabs-Internship

## Description
This repository contains my Java-based development projects completed during the DecodeLabs internship program. 

**Project 1: Number Guessing Game**
A Java console application that challenges users to guess a randomly generated number. This project focuses on logic loops, robust defensive programming to handle invalid user inputs gracefully, and session persistence to accurately track gameplay states.

**Project 2: Student Grade Calculator**
A modular, Object-Oriented Java application that processes student scores across an arbitrary number of subjects. It features strict input validation (0-100 bounds), exception handling for type mismatches, and automated computation of total marks, averages, and final letter grades.

## How to Run
1. Ensure you have the Java Development Kit (JDK) installed on your machine.
2. Clone this repository to your local machine.
3. Open the repository folder in NetBeans IDE.
4. Navigate into the specific project folder you wish to test (e.g., `DecodeLab1` or `DecodeLab2`).
5. Click the **Run Project** button (the green play arrow) in the top toolbar, or press `F6`, to start the application in the console.
