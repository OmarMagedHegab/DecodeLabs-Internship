# Number Guessing Game

## Description
A Java-based application that challenges users to guess a randomly generated number. This project focuses on robust code architecture, featuring defensive programming techniques to handle invalid user inputs gracefully, alongside session persistence to accurately track and maintain gameplay states.

## How to Run
1. Ensure you have the Java Development Kit (JDK) installed on your machine.
2. Clone this repository to your local machine.
3. Open the project folder in NetBeans IDE.
4. Click the **Run Project** button (the green play arrow) in the top toolbar, or press `F6`, to start the application in the console.
